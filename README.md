# CS300 Assignment 4 - Book Rating App


A React + Vite book rating app where you can:

- search books from Open Library
- rate books with stars
- save rated books in a dedicated Rated Books tab
- remove individual rated books or clear all

## GitHub Pages URL

For a project repo named `CS300-Assignment4`, the deployed site URL is:

`https://<your-github-username>.github.io/CS300-Assignment4/`

Example:

`https://brandonmariobahena.github.io/CS300-Assignment4/`

## Local development

```bash
npm install
npm run dev
```

## Production build
